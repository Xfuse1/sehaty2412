# **App Name**: Sehaty

## Core Features:

- Digital Pharmacy Communication: Allow users to communicate with pharmacies via WhatsApp for prescriptions and product inquiries.
- Physiotherapy Package Booking: Enable users to view available physiotherapy packages, book appointments, and make payments through the app.
- Surgery Consultations and Booking: Facilitate pre-surgery consultations and surgery booking, including a specialized section for cosmetic surgeries and hair transplants.
- Appointment Management: Allow users to book, reschedule, and manage appointments with doctors and clinics.
- Home Nursing Care Booking: Enable users to book home nursing care sessions with monthly/weekly packages.
- Home Checkup Requests: Allow users to request home medical visits and remote initial diagnoses.
- Lab Services Booking and Results: Enable users to book sample collection visits and access digital analysis results.

## Style Guidelines:

- Primary color: Soft blue (#75A2C6) to convey trust and tranquility.
- Background color: Light blue (#E8F0F7) to create a calm and clean environment.
- Accent color: Light purple (#A283B3) to highlight important elements and calls to action.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and accessible look, ensuring readability for all users.
- Use clear and large icons representing each service, ensuring easy recognition for users with varying tech experience.
- Employ a simple and clear design to cater to users with low tech experience. Minimize footer elements for a cleaner interface.
- Use subtle animations for transitions and loading states to improve user experience without overwhelming users.
